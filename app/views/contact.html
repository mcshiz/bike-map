• The trails were mapped using the My Tracks app for android.
    • Install the app
    • Ride a trail while tracking your position
    • Export the trail file to Google Drive or external storage as a .KML file
    • Send the KML file with the button below or send a link to a public Google Drive file
  • Exporting Strava data
    • Ride a trail while tracking your position
    • Save your ride to Activity Feed
    • Log into your account on Strava.com
    • Select the ride you wish to share
    • Click the wrench icon and Export GPX
    • Send the GPX file with the button below
  • When sending a trail file please send it as .KML or .KMZ if possible
